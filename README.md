# catalogo-jogos-python-sql
Projeto: Catálogo de Jogos (Python + SQL)  Aplicação de console em Python para gerenciar uma coleção de videogames, demonstrando o uso prático de SQL.  Tecnologias Utilizadas:  Python  SQL (SQLite)  Funcionalidades (CRUD):  Adicionar jogos (INSERT)  Listar todos os jogos (SELECT)  Atualizar status de um jogo (UPDATE)  Deletar um jogo (DELETE)
